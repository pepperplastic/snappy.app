# Snappy — snappy.gold

Instant offers for jewelry, watches & precious metals. Snap a photo, get an offer.

## Deploy to Vercel

1. Push to GitHub
2. Import in Vercel
3. Add env var: `ANTHROPIC_API_KEY`
4. Deploy

## Run locally

```bash
npm install
npm run dev
```

## Custom domain

In Vercel Settings → Domains, add:
- snappy.gold (primary)
- snappygold.com (redirect to snappy.gold)
